# Akshitha R - Personal Portfolio

A professional and responsive personal portfolio website designed to showcase my projects, skills, education, and internship experiences. Built using modern web technologies to be visually appealing, user-friendly, and fully responsive across all devices.

## 🚀 Features

- **Responsive Design**: Adapts seamlessly to desktops, tablets, and mobile devices.
- **Dynamic Navigation**: Sidebar navigation for desktop and top-bar for mobile, with smooth scrolling.
- **Project Showcase**: Highlights key projects such as the Intelligent Pesticide Sprinkler, College Conference Website, and Smart Wheelchair simulation.
- **Experience & Education**: Details of my internships and academic journey, including my minor in Business Data Analytics.
- **Skills Section**: Organized technical skills ranging from web development to IoT tools.
- **Downloadable Resume**: Direct link to download my latest PDF resume.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure
- **CSS3**: Custom styling, animations, flexbox, grid, and responsive layouts
- **JavaScript (ES6+)**: Interactive elements, scroll spying, and smooth transitions
- **Font Awesome**: Icons for UI elements
- **Google Fonts**: Modern typography (Outfit)

## 💻 Local Installation & Setup

To view this portfolio locally on your machine, simply follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Akshi-hub23/Portfolio_resume.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd Portfolio_resume
   ```

3. **Open the project:**
   Simply open the `index.html` file in your preferred web browser, or use a local development server like VS Code's "Live Server" extension for a better development experience.

## 📁 Repository Structure

- `index.html`: The main HTML document containing the structure of the portfolio.
- `style.css`: The stylesheet containing all styling rules and responsive media queries.
- `script.js`: The JavaScript file handling navigation, animations, and interactive elements.
- `Akshitha_Resume.pdf`: Downloadable copy of my professional resume.
